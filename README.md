# Habit Tracker (Android App)

Ένας μινιμαλιστικός, offline οδηγός καταγραφής συνηθειών (Habit Tracker) για συσκευές Android, εμπνευσμένος από τη σχεδίαση παραδοσιακού χάρτινου μπλοκ σημειώσεων.

Η εφαρμογή λειτουργεί αποκλειστικά σε οριζόντια προβολή (landscape) και δεν περιέχει διαφημίσεις, προσφέροντας μια καθαρή και απλή εμπειρία χρήσης.

## Χαρακτηριστικά (Features)

* **Μινιμαλιστική Αισθητική:** Υπόβαθρο με υφή και χρώμα χαρτιού (warm beige), λεπτές γραμμές και χρήση της χειρόγραφης γραμματοσειράς *Ink Free*.
* **Δυναμικό Πλέγμα 31 Ημερών:** Εμφάνιση 31 κύκλων για κάθε συνήθεια, ευθυγραμμισμένων με τους αριθμούς των ημερών.
* **Τοπική Αποθήκευση (Room DB):** Όλα τα δεδομένα αποθηκεύονται τοπικά στη συσκευή. Δεν απαιτείται σύνδεση στο διαδίκτυο ή δημιουργία λογαριασμού.
* **Ιστορικό Ετών & Μηνών:** Δυνατότητα πλοήγησης σε διαφορετικά έτη και επιλογή μήνα με κόκκινο highlight για τον τρέχοντα.
* **Δυναμική Διαχείριση:** Εύκολη προσθήκη νέων συνηθειών και γρήγορη διαγραφή με το πάτημα ενός "x".
* **Εξαγωγή Δεδομένων (Export to PC):** Δημιουργία αρχείου `.csv` με τις καταγραφές του τρέχοντος μήνα και άμεση κοινή χρήση (share) για εύκολη ανάγνωση στο Excel του υπολογιστή σας.

## Τεχνολογίες (Tech Stack)

* **Γλώσσα Ανάπτυξης:** Kotlin
* **Αρχιτεκτονική:** MVVM (Model-View-ViewModel)
* **Βάση Δεδομένων:** Room Database (SQLite) για τοπική αποθήκευση
* **Ασύγχρονος Προγραμματισμός:** Kotlin Coroutines & Flow για άμεση, αντιδραστική (reactive) ανανέωση της οθόνης
* **Σχεδίαση UI:** XML Layouts (ConstraintLayout, RecyclerView) με Custom XML Drawables για τους κύκλους
